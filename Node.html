Що таке Node.js

Node.js — це середовище виконання JavaScript, яке дозволяє запускати JavaScript не тільки в браузері, а й на сервері. Завдяки Node.js можна створювати серверну частину сайту, працювати з файлами, базами даних і обробляти запити від користувачів. Раніше JavaScript використовувався тільки для роботи на сторінці в браузері, але Node.js дозволив використовувати його для створення повноцінних веб-додатків, де є і клієнтська частина, і сервер.

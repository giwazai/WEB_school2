А навіщо тоді JS?

JavaScript — це мова програмування, яка використовується для того, щоб зробити сайт інтерактивним та динамічним. За допомогою JavaScript можна додавати різні дії на сторінці, наприклад: відкривати меню, змінювати текст, реагувати на натискання кнопок, перевіряти форми, створювати анімації та працювати з даними. Якщо HTML створює структуру сайту, а CSS відповідає за його вигляд, то JavaScript відповідає за поведінку сайту. Завдяки JavaScript сайт може реагувати на дії користувача без перезавантаження сторінки. Наприклад, можна зробити слайдер, модальне вікно, випадаюче меню, перевірку пароля або завантаження даних з сервера. JavaScript працює разом з HTML і CSS та є однією з трьох основних технологій веб-розробки.

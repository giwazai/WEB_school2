А для чого CSS?

CSS (Cascading Style Sheets) — це мова стилів, яка використовується для оформлення веб-сторінки. Вона відповідає за те, як саме виглядають елементи сайту, які були створені за допомогою HTML. За допомогою CSS можна змінювати колір тексту, розмір шрифту, відступи, фон, розташування елементів, додавати анімації, тіні, рамки та робити сайт адаптивним для різних екранів. CSS потрібен для того, щоб сайт був красивим, зручним і сучасним, тому що без стилів сторінка буде виглядати як звичайний текст без оформлення. CSS працює разом з HTML: HTML створює структуру, а CSS задає зовнішній вигляд цієї структури.
